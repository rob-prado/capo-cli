# Skill: Bootsplash Sandbox Generation

## Business Value
This skill executes the generation of React Native bootsplash (launch screen) native artifacts (Android/iOS) while strictly enforcing the **Sandbox Strategy**. It ensures that running `react-native-bootsplash` does not permanently pollute the base repository structure (`android/app/src/main/res` or standard iOS project folders) with brand-specific images or colors.

## Expected Inputs
The script (`scripts/generate-splash.sh`) expects:
- `PROJECT_DIR`: The absolute path to the initialized project.
- `BRAND`: The target brand name.
- `LOGO_PATH`: Absolute or relative path to the PNG logo.
- `BG_COLOR`: Hex color code for the splash screen background.
- `FLAVOR_1` [FLAVOR_2...]: A dynamic list of target flavors (e.g., `dev staging prd`) where the generated native artifacts should be permanently distributed.

## Execution Flow
1. **Sandbox Creation**: Backs up the clean agnostic state of `android/app/src/main/res` and `AndroidManifest.xml` into a hidden `.bootsplash_backup` directory.
2. **Library Execution**: Runs `npx react-native-bootsplash generate` against the provided logo and background color. This library blindly mutates `main/res` and `ios/` folders.
3. **Android Flavor Routing**:
   - Safely copies the newly generated `drawable-*` and `mipmap-*` folders, plus `colors.xml` and `styles.xml`, directly into the provided flavor targets (e.g., `android/app/src/dev/res`).
   - Renames `bootsplash_logo.png` to `logo.png` for standardization.
4. **iOS Name Refinement**:
   - Intercepts Xcode artifacts generated by the library (`BootSplash.storyboard`, `BootSplashLogo-*.imageset`) and securely renames them to standard `LaunchScreen` equivalents to match standard Apple paradigms.
   - Patches `Info.plist`, `.pbxproj`, and JSON definitions to remap the renamed assets.
5. **Sandbox Destruction & Cleanup**:
   - Wipes the tainted `main/res` folder.
   - Restores `main/res` and `AndroidManifest.xml` from the backup.
   - Deletes leftover library artifact folders (e.g., `assets/bootsplash`).
